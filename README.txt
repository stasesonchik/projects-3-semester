Описание

Скрипт генерирует синтетический временной ряд (даты + случайные значения) и строит его график.
Используются библиотеки pandas, numpy, matplotlib.

Как запустить:
pip install pandas numpy matplotlib
python DA-1-14.py


Функции
generate_time_series(...)
  Создаёт DataFrame с колонками:
  date — даты,
  value — случайные значения.
  Параметры: начальная дата, количество точек, частота (D, H, M), диапазон значений.

plot_time_series(data, title)
  Строит график временного ряда. Проверяет, что в данных есть date и value.


Результат: График со случайными значениями по Y c равномерным распеределением и временем по X.
